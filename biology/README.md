# Biology

This package will house the biology execution layer, including base-units, polymer interactions, and reaction systems that can interpret and transform genome outputs into active processes. It is intended to stay modular so different execution chemistries can be explored without coupling to the simulator runtime.

Planned components:
- Base-unit definitions and composition rules
- Polymer and sequence interaction models
- Reaction primitives (bind, copy, cut, release)
- Resource pools, decay, and recycling helpers
- Unit tests for interaction and reaction behavior
