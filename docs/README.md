# Documentation

The documentation space collects architecture notes, biology and genetics specifications, and development logs. Use this area to capture design decisions, experiment results, and outlines for future work.

Suggested subfolders:
- `architecture/` for diagrams and biology/simulator module overviews
- `genes/` for gene, regulatory, or heredity specifications
- `logs/` for dev journals and experiment notes
- `references/` for external research links and summaries
